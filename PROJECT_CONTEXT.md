# PROJECT CONTEXT

## Firma
Fleetware management, monitoring vozidel - poloha v realném čase, knihy jízd, 
Odpadová vozidla - monotoring výsypů, vážení vysypaného materiálu. 
disoečerské řízení - trasy objednávek,navigace do vozidel dle trasy

## Produkty
FLEETWARE - evidence vozidel, informace z vozidel - km, ridič, aktivace vstupů, spotřeby, poloha 
ROAD PLAN - plánování objednaných služeb pro konkrétního zákazníka, plánování na vozidlo, průběh výkonu v reálném čase
PASPORT - evidence nádob, smluv, evidence výsypů, evidence stanovišť nádob
## Technologie
GPS/GLONASS LOKACE,GSM/LTE - komuikace, DPE STACK, SQL DB, samostatná DB každý produkt

## Architektura systému
FLEETWARE - práva na jednotlivé systémy včetně RoadPlan a PASPORT, ve FLEETVARE vznikají poziční data a události

## Integrace
REST API
